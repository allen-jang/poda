# poda

Posting dashboard static preview for copy-ready Naver Blog articles.

- `index.html` lists prepared articles and copy controls.
- `articles/` contains per-article detail pages.
- `assets/` contains generated images used in previews.
- `data/articles.json` is the generated article index.

This site is a review/copy surface only. It includes `noindex` robots meta tags and does not include Naver login, publishing, engagement, affiliate, or analytics automation controls.
